# Relación de aspecto de Imagenes

Tabla con los formatos más comunes y recomendados para SEO y para blogs para imagenes.

|Uso Principal|Relación de Aspecto|Ancho x Alto|Descripción|
|-------------|-------------------|------------|-----------|
|Imagen Destacada|16:9|1200 x 675 píxeles|Estándar para blogs. Se adapta bien a redes sociales y previsualizaciones.|
|Imagen dentro del Contenido|16:9 o 3:2|800 x 450 píxeles|Mantén la coherencia con la imagen destacada. No más ancha que el contenedor de texto.|
|Banner Hero / Cabecera|21:9 (Ultra Wide) o 3:1|1920 x 820 píxeles|Para un impacto visual fuerte en la parte superior del artículo. Usar con moderación.|
|Imagen para Redes Sociales (Open Graph)|1.91:1 (Facebook/LinkedIn)|1200 x 630 píxeles|CRÍTICO para SEO. Esta es la relación que Facebook y LinkedIn usan para previsualizar enlaces.|
|Imagen para Twitter Card|2:1|1200 x 600 píxeles|Si quieres optimizar específicamente para Twitter.|
|Imagen Cuadrada (Instagram/Thumbnail)|1:1|1080 x 1080 píxeles|Útil si piensas republicar el contenido en Instagram o para miniaturas en galerías.|
|Imagen Vertical (Pinterest/Mobile)|4:5 o 2:3|1000 x 1500 píxeles|Gana popularidad por el consumo en móviles. Ideal para tutoriales visuales.|
